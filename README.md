# Dota Heroes

an app that shows Stats of Dota 2 Heroes

## Installation

1. After cloning the project, open your terminal and run this line 
```
pod install
```

2. Open the .xcworkspace file

### Built with
- **Swift 5**
- **VIP Swift**
- **RxSwift** + **RxDataSources**
- **Realm**
- **Alamofire**
- **Snapkit**
- **API from: https://api.opendota.com**
